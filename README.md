# 불교학 학술지 편집검토 프롬프트 아카이브

> 투고규정 PDF와 AI 편집검토 프롬프트를 함께 제공하는 학술지 편집 보조 도구

**제작일** 2026년 4월 29일  
**배포** GitHub Pages (정적 웹앱)

---

## 개요

이 저장소는 불교학 관련 학술지에 투고 예정인 논문을 AI 도구로 형식 검토할 수 있도록 지원하는 정적 웹앱입니다.  
학술지별 투고 양식 PDF와 AI 편집검토 프롬프트를 한 곳에서 내려받을 수 있으며, Claude·Gemini·ChatGPT 각각의 사용 절차를 안내합니다.

이 앱은 서버 없이 작동하며, 사용자의 파일을 외부로 전송하거나 저장하지 않습니다.

---

## 주요 기능

- 29종 불교학 관련 학술지 투고 양식 PDF 제공
- 불교학 학술지 전용 AI 편집검토 프롬프트 다운로드 및 자동 복사
- 범용 인문사회 학술지 편집검토 프롬프트 다운로드 및 자동 복사
- Claude / Gemini / ChatGPT 도구별 사용 절차 안내
- 사용법 안내 영상 임베드
- 모바일 반응형 디자인

## 파일 구조

```
/
├─ index.html
├─ assets/
│  ├─ css/style.css
│  ├─ js/app.js
│  └─ pdfs/          ← 학술지별 투고 양식 PDF (29개)
├─ prompts/
│  ├─ claude_buddhist_journal_editorial_review_prompt_ultimate.md
│  └─ claude_journal_editorial_review_prompt_generic.md
├─ README.md
└─ LICENSE
```

---

## 배포 방법

1. 이 저장소를 GitHub에 업로드합니다.
2. **Settings → Pages → Branch: main** 으로 설정합니다.
3. `assets/pdfs/` 폴더에 투고 양식 PDF 파일을 넣습니다.
4. `prompts/` 폴더에 편집검토 프롬프트 `.md` 파일을 넣습니다.
5. `assets/js/app.js` 상단의 `YOUTUBE_VIDEO_ID` 값을 안내 영상 ID로 교체합니다.

## 사용 절차 요약

| 단계 | 내용 |
|------|------|
| 1 | 학술지 선택 후 투고 양식 PDF 다운로드 |
| 2 | 편집검토 프롬프트 다운로드 (자동 클립보드 복사) |
| 3 | AI 도구에 투고 양식 PDF + 논문 PDF 업로드 |
| 4 | 투고규정 분석 및 저널 전용 편집 기준표 작성 요청 |
| 5 | 복사된 프롬프트 붙여넣기 후 논문 전수 검토 요청 |
| 6 | Claude: docx 출력 / Gemini: Google Docs 내보내기 / ChatGPT: Excel 출력 |

## 한글 파일명 오류 시

GitHub Pages에서 한글 파일명이 작동하지 않는 경우:

1. `assets/pdfs/` 내 PDF 파일명을 영문으로 변경합니다.  
   예: `선학.pdf` → `sunhak.pdf`
2. `assets/js/app.js`의 `journals` 배열에서 해당 항목의 `file` 값을 동일하게 수정합니다.

## 주의사항

- AI 검토 결과는 참고자료이며, 최종 판단은 연구자·편집자가 직접 수행해야 합니다.
